### Prompt Templates
#### Portrait
```
[Subject] with [realistic skin texture], [specific lighting], [camera details], photorealistic, 8K, HDR. Avoid: plastic skin, blurry details.
```

#### Product Photography
```

```

#### Landscape
```
[Scene] at [time of day], [specific lighting], [camera details], volumetric atmosphere, hyper-detailed textures, cinematic color grading, 8K, HDR. Avoid: oversaturated colors, cartoonish elements.
```

### Key Techniques
1. **Lighting**: Always specify time of day and light quality (e.g., "golden hour", "volumetric fog")
2. **Textures**: Describe materials with realistic properties (e.g., "rough concrete", "glossy ceramic")
3. **Depth of Field**: Use terms like "shallow DOF" or "deep focus"
4. **Color Grading**: Specify color tones (e.g., "teal and orange color grading")
5. **Consistency**: For edits, use "Strictly preserve facial identity, skin texture, and body structure"
